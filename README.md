# World of Tales

A gentle, child-friendly fairy-tale adventure for **ages 6+**. Explore an
enchanted forest, build a cosy home, craft potions, grow a garden, and rescue
eggs to hatch and tame your very own pets — a fox, a baby dragon, a unicorn,
and more. Everyone is kind, the world is safe, and there's no chat.

## ▶ Play now (in your browser)

### https://stefanzvonar.github.io/world-of-tales-play/

Runs on desktop, tablet, and phone — no install needed. The first load is
around 50 MB, so give it a moment the first time. Tap the **⛶** button to go
fullscreen, or **Add to Home Screen** on a tablet/phone for an app-like
fullscreen experience.

## Controls

**Desktop**
- **Move** — W A S D
- **Look around** — hold the right mouse button and move
- **Interact / gather / rescue** — E
- **Jump** — Space
- **Bag / Craft / Build / Menu** — on-screen buttons

**Touch (tablet / phone)**
- **Move** — left on-screen joystick
- **Look around** — drag the right side of the screen
- **Use** and **Jump** — on-screen buttons
- Hold your device in **landscape** for the best view

## What you can do
- Create a character — human, elf, dwarf, mermaid, witch, princess, prince, king or queen
- Explore a storybook forest with blossom trees, lakes, and friendly (and cheeky) creatures
- Gather wood, petals, herbs, and magic crystals
- Craft tools, clothes, potions, and spells
- Build houses and decorations, and plant a garden
- Shoo mischievous goblins to save eggs, then hatch and bond with pets

## About
A stylized 3D browser game built with the [Godot Engine](https://godotengine.org).
This repository hosts the compiled web build, which updates as the game grows.
